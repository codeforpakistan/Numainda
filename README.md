# Numainda

## Overview
Numainda is a chatbot designed to provide information about Pakistan's National Assembly sessions based on the bulletins published on NA's website. It employs advanced language processing technologies to deliver insightful and accurate responses to user queries.

## Key Features
- Interactive chatbot interface for easy user interaction.
- Real-time insights from National Assembly sessions.
- Sophisticated natural language understanding for accurate query responses.

## Built With
- [Chainlit](https://docs.chainlit.io/) - For building interactive chat interfaces.
- [Langchain](https://github.com/rubentak/Langchain) - Utilized for integrating large language models.
- [LlamaIndex](https://docs.chainlit.io/integrations/llama-index) - For efficient data indexing and retrieval.
- [OpenAI](https://www.openai.com/) - Leveraging GPT models for natural language processing.

## Getting Started

### Installation
To install Numainda, clone the repository and install the necessary dependencies:
```bash
git clone https://github.com/yourusername/numainda.git
cd numainda
pip install -r requirements.txt
```

## Running the Application
Use the following command to run Numainda:

```bash
chainlit run main.py -w
```
